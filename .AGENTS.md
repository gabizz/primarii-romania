## STEP 2: Enrich with CIF/CUI from DPFBL
- **Source:** `http://www.dpfbl.mdrap.ro/harta_judete_cont_unic.html`
- **Action:**
  1. Write a scraper that navigates/fetches data from the DPFBL portal. Notice that the data might be split by county (judet).
  2. Match the county (`judet`) and locality name (`nume`) from the Step 1 CSV with the records on DPFBL. Use fuzzy matching if exact string matching fails due to diacritics or prefixes (e.g., "Comuna X" vs "X").
  3. Extract the `CIF` (Cont Unic/CUI) for each matched UAT.
  4. Append the CUI as a new property (`cui`) to the corresponding object in the dataset.
  5. Log any UATs where the CUI could not be matched for manual review.

## STEP 3: ANAF API Batch Processing & Address Enrichment
- **Source:** ANAF TVA API `https://webservicesp.anaf.ro/api/PlatitorTvaRest/v9/tva`
- **Action:**
  1. Filter the dataset to include only records that successfully received a `cui` in Step 2.
  2. Chunk the dataset into **batches of maximum 50 elements** to respect rate limits and the specific batching requirement.
  3. For each chunk, map the elements to the ANAF payload structure: `[{ cui: "...", data: "YYYY-MM-DD" }, ...]`. Use the current date.
  4. Send a `POST` request to the ANAF API for each batch.
  5. Parse the `result.found` array from the ANAF response. Extract the full official address (e.g., `date_generale.adresa` or equivalent field).
  6. Merge this official address back into the main dataset.
  7. Implement a delay (sleep) between batch requests to avoid API throttling.

**Reference Script for ANAF Integration:**
Use the logic from the following implementation as a baseline for the API call structure in Step 3:

```javascript
import moment from "moment"

const ANAF_ENDPOINT = "[https://webservicesp.anaf.ro/api/PlatitorTvaRest/v9/tva](https://webservicesp.anaf.ro/api/PlatitorTvaRest/v9/tva)"

// Example wrapper for the agent to adapt for batch processing
async function fetchAnafDataBatch(cuiArray) {
    const dt = moment().format("YYYY-MM-DD")
    
    // cuiArray should be an array of up to 50 CUIs
    const payload = cuiArray.map(cui => ({
        cui: cui,
        data: dt
    }))
        
    try {
        const response = await fetch(ANAF_ENDPOINT, {
            method: "POST",
            body: JSON.stringify(payload),
            headers: {
                "Content-Type": "application/json"
            }
        })
        const result = await response.json()
        
        let enrichedData = {}
        if (result.found) {
            // Adapt this reduce function to also grab the address field
            enrichedData = result.found.reduce((acc, el) => ({
                ...acc, 
                [el.date_generale.cui]: {
                    denumire: el.date_generale.denumire,
                    adresa: el.date_generale.adresa // Agent: verify exact key for address in ANAF response
                }
            }), {})
        }
        return enrichedData
    } catch (error) {
        console.error("ANAF API Error:", error)
        return null
    }
}
